Project Title :: HOSTEL MANAGEMENT SYSTEM Using PHP and MySQL

Project By :: Chaitanya Kini

1. Install XAMPP on your local system.
2. Download and Unzip the Project File on your local system.
3. Put the 'hostel' file inside xampp/htdocs/ .
4. Database Configuration
    a. Open phpmyadmin by typing 'localhost/phpmyadmin/' in your Browser or By clicking 'Admin' in XAMPP Application
    b. Create Database 'hostel'
    c. Import database 'hostel.sql' file
5. Open Your Browser and type 'localhost/hostel/'
6. Hompage will be displayed - Click on the User Registration and Fill the details.
